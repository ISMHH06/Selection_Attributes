3Wrapper : Sélection Ascendante

Forward Selection. On commence avec 0 attribut et on ajoute itérativement l'attribut qui améliore le plus les performances croisées d'un classifieur KNN(k=5).

from sklearn.neighbors import KNeighborsClassifier
from sklearn.feature_selection import SequentialFeatureSelector

knn = KNeighborsClassifier(n_neighbors=5)

sfs_forward = SequentialFeatureSelector(
    estimator=knn, n_features_to_select=10, 
    direction='forward', cv=5
)
sfs_forward.fit(X_train, y_train)
forward_mask = sfs_forward.get_support()

4Wrapper : Sélection Descendante

Backward Elimination. On commence avec les 64 attributs et on retire itérativement l'attribut dont l'absence dégrade le moins l'accuracy du KNN.

Complexité algorithmique : Forward démarre de 0 et fait n itérations. Backward démarre de 64 features et doit évaluer (64 × 63 × ... × 54) combinaisons avec validation croisée à chaque étape → O(n²) évaluations contre O(n) pour Forward. 
sfs_backward = SequentialFeatureSelector(
    estimator=knn, n_features_to_select=10, 
    direction='backward', cv=5
)
sfs_backward.fit(X_train, y_train)
backward_mask = sfs_backward.get_support()

Validation des approches en entraînant un KNN final sur X_train réduit et test sur X_test réduit. Comparaison avec la baseline à 64 attributs.

def eval_subset(mask):
    clf = KNeighborsClassifier(n_neighbors=5)
    clf.fit(X_train[:, mask], y_train)
    return clf.score(X_test[:, mask], y_test)
